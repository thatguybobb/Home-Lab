# Home-Lab
Cybersecurity Homelab

Hello! This repository documents my hands-on cybersecurity home lab, where I simulate real-world enterprise environments to develop skills in system administration, network security, and threat detection.

This lab builds on both my coursework and personal interest in cybersecurity, and focuses on gaining practical, real-world experience through building, breaking, and securing systems.

## 🧠 Objective
The goal of this lab is to build practical experience with:
- Active Directory and Windows domain environments
- Network configuration and segmentation
- Security monitoring and log analysis
- Attack simulation and detection

---

## 🧱 Lab Architecture

- **pfSense** – Firewall and network segmentation  
- **Windows Server 2022** – Domain Controller (AD, DNS, DHCP)  
- **Windows 11** – Domain-joined client machine  
- **Kali Linux** – Attacker machine for simulations  
- **Wazuh (SIEM)** – Log collection and threat detection *(Phase 2)*  
- **Snort (IDS)** – Network intrusion detection  

---

## 🚀 Project Phases

### Phase 1: Active Directory Environment
- Configure Windows Server as a Domain Controller  
- Set up DNS and DHCP  
- Create users, groups, and Group Policy Objects (GPOs)  
- Join Windows 11 client to the domain  

### Phase 2: Security Monitoring (SIEM)
- Deploy Wazuh SIEM  
- Install agents on endpoints  
- Monitor logs and analyze security events  

### Phase 3: Attack Simulation & Detection
- Perform network scans and attacks using Kali Linux  
- Generate and analyze alerts in SIEM and IDS tools  
- Investigate suspicious activity and correlate logs
